еңЁ JS жЁЎеқ—еҢ–зј–зЁӢдёӯпјҢд№ӢеүҚдҪҝз”Ёзҡ„жҳҜ require.js жҲ–иҖ… sea.jsгҖӮйҡҸзқҖеүҚз«Ҝе·ҘзЁӢеҢ–е·Ҙе…· webpack зҡ„жҺЁеҮәпјҢдҪҝеҫ—еүҚз«Ҝ js еҸҜд»ҘдҪҝз”Ё CommonJS жЁЎеқ—ж ҮеҮҶжҲ–иҖ…дҪҝз”Ё ES6 moduel зү№жҖ§гҖӮ
еңЁ CommonJs жЁЎеқ—ж ҮеҮҶдёӯжҲ‘д»¬иҪҪе…ҘжЁЎеқ—дҪҝз”Ёзҡ„жҳҜ require()пјҢиҫ“еҮәжЁЎеқ—з”Ёзҡ„жҳҜ exports жҲ–иҖ… module.exports
еңЁ ES6 дёӯиҪҪе…ҘжЁЎеқ—жҲ‘д»¬з”Ёзҡ„жҳҜ import пјҢиҫ“еҮәжЁЎеқ—з”Ёзҡ„жҳҜ export

exports дёҺ module.exports зҡ„еҢәеҲ«

//иҪҪе…ҘжЁЎеқ—
var m = require('./moduleA.js')
m.callName()
//иҫ“еҮәжЁЎеқ—
exports.callName = function(){
    console.log('jesse')
}
//д№ҹеҸҜд»Ҙиҝҷж ·иҫ“еҮә
module.exports.callName = function(){
    console.log('jesse')
}

module.exports жүҚжҳҜ module жЁЎеқ—зҡ„зңҹжӯЈжҺҘеҸЈпјҢиҖҢ exports еҸҜд»ҘзҗҶи§Јдёәе®ғзҡ„дёҖдёӘеүҜжң¬
иҷҪз„¶дҝ®ж”№ exports еҜ№иұЎзҡ„ж—¶еҖҷд№ҹдјҡдҝ®ж”№ module.exports еҜ№иұЎпјҢдҪҶжңҖз»Ҳиҝ”еӣһз»ҷи°ғз”Ёзҡ„жҳҜ module.exports еҜ№иұЎ
еҪ“ module.exports еҜ№иұЎйҖҡиҝҮиөӢеҖјж–№ејҸиҝӣиЎҢи®ҫе®ҡеҗҺпјҢе°ұе·Із»Ҹе’Ң exports еҜ№иұЎжІЎе…ізі»дәҶ

so,жҲ‘зҡ„зҗҶи§ЈжҳҜ exports иҫ“еҮәзҡ„еҸӘжҳҜиҰҒиҫ“еҮәзҡ„еҜ№иұЎзҡ„жҹҗдёӘеұһжҖ§пјҢmodule.exports жүҚзңҹжӯЈиҫ“еҮәзҡ„жҳҜиҰҒиҫ“еҮәзҡ„еҜ№иұЎгҖӮ
дёҖдёӘжЁЎеқ—ж–Үд»¶дёӯеҸҜд»ҘжңүеӨҡдёӘ exports иҫ“еҮәпјҢдҪҶеҸӘиғҪжңүдёҖдёӘ module.exports иҫ“еҮә
иҝҷз§Қж–№ејҸжҷ®йҒҚз”ЁдәҺ node дёӯжЁЎеқ—зҡ„зј–еҶҷ

export дёҺ export.defult еҢәеҲ«

es6moduel зү№жҖ§еңЁ node зҺҜеўғдёӯе№¶дёҚиғҪе®Ңе…Ёж”ҜжҢҒпјҢи§ЈеҶіж–№жі•жҳҜз”Ё babel зј–иҜ‘

//иҪҪе…ҘжЁЎеқ—
import {callName} from './moduleA.js'
callName()
//иҫ“еҮәжЁЎеқ—
export function callName (){
    console.log('jesse')
}

1.export дёҺ export default еқҮеҸҜз”ЁдәҺеҜјеҮәеёёйҮҸгҖҒеҮҪж•°гҖҒж–Үд»¶гҖҒжЁЎеқ—зӯү 2.дҪ еҸҜд»ҘеңЁе…¶е®ғж–Үд»¶жҲ–жЁЎеқ—дёӯйҖҡиҝҮ import+(еёёйҮҸ | еҮҪж•° | ж–Үд»¶ | жЁЎеқ—)еҗҚзҡ„ж–№ејҸпјҢе°Ҷе…¶еҜје…ҘпјҢд»ҘдҫҝиғҪеӨҹеҜ№е…¶иҝӣиЎҢдҪҝз”Ё 3.еңЁдёҖдёӘж–Үд»¶жҲ–жЁЎеқ—дёӯпјҢexportгҖҒimport еҸҜд»ҘжңүеӨҡдёӘпјҢexport default д»…жңүдёҖдёӘ 4.йҖҡиҝҮ export ж–№ејҸеҜјеҮәпјҢеңЁеҜје…Ҙж—¶иҰҒеҠ { }пјҢexport default еҲҷдёҚйңҖиҰҒ

ж–Үз« зӣ®еҪ•

  2. export дёҺ export.defult еҢәеҲ«
